# 🇮🇳 JanSeva – Centralized Public Grievance Portal (Prototype)

JanSeva is a modern, responsive, front-end prototype of India's centralized public grievance system.  
It allows users to register, log in, raise complaints, and track complaint status.

This project is built using **HTML, CSS, and JavaScript only** with no backend.  
All data is stored in **localStorage**.

---

## 🚀 Live Features

### 🏠 1. Sticky Navigation Bar
- Smooth scrolling navigation
- Responsive design
- FontAwesome icons
- Glassmorphism effect

### 🎨 2. Premium Animated Background
- Moving tricolor gradient (Saffron, White, Green, Blue)
- Floating blurred glow blobs
- Particle animation overlay
- Modern glass UI

### 👤 3. User Registration
- Full Name
- Email
- Phone
- Address
- Password
- Stores user data in `localStorage`
- Shows success alert
- Clears form after submission

### 🔐 4. User Login
- Email & Password validation
- Verifies data from `localStorage`
- Displays welcome message
- Shows success/failure alerts

### 📝 5. Raise Complaint
- Category selection:
  - Roads
  - Water
  - Electricity
  - Garbage
  - Other
- Description
- Location
- Image upload with preview
- Generates Random Complaint ID
- Stores complaint in `localStorage`
- Displays generated Complaint ID

### 🔎 6. Track Complaint
- Search using Complaint ID
- Displays:
  - Category
  - Description
  - Location
  - Status (Default: Pending)

### 📊 7. Complaint Counter
- Dynamic complaint counter
- Updates automatically after submission

### ✨ 8. Scroll Animations
- Smooth fade-in effects
- Modern UI transitions

### 📱 9. Fully Responsive
- Mobile compatible
- Desktop optimized
- Clean and modern layout
